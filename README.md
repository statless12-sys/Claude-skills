1. /Tender Analyzer
محلل المناقصات
Function:
Reviews tender documents and identifies risks, missing information, contractual concerns, clarification questions, and bid recommendations before submission.
الوظيفة:
يقوم بمراجعة مستندات المناقصة وتحديد المخاطر والمعلومات الناقصة والملاحظات التعاقدية والأسئلة المطلوبة للاستفسار والتوصيات الخاصة بتقديم العرض قبل التقديم النهائي.
Typical Output Format:
•	Executive Summary
•	Tender Risk Register
•	Missing Information List
•	Clarification Questions
•	Bid Recommendation
تنسيق المخرجات المتوقع:
•	ملخص تنفيذي
•	سجل مخاطر المناقصة
•	قائمة المعلومات الناقصة
•	أسئلة الاستفسار والتوضيح
•	توصية تقديم العرض
 
2. /BOQ Reviewer
مراجع جداول الكميات

الوظيفة:
يقوم بمراجعة جداول الكميات لاكتشاف البنود الناقصة أو الأخطاء في الكميات أو البنود المكررة أو فجوات نطاق العمل أو مخاطر التسعير وفرص تحسين الربحية.
Typical Output Format:
•	BOQ Review Summary
•	Missing Items Table
•	Quantity Inconsistency Table
•	Pricing Risk Notes
•	Recommended Corrections
تنسيق المخرجات المتوقع:
•	ملخص مراجعة جدول الكميات
•	جدول البنود الناقصة
•	جدول اختلافات الكميات
•	ملاحظات مخاطر التسعير
•	التصحيحات المقترحة

•	سجل مخاطر العقد
•	التزامات المقاول

8. /Critical Path Analyzer
محلل المسار الحرج
Function:
Reviews project schedules and identifies critical path activities, near-critical activities, float risks, missing logic links, scheduling weaknesses, unrealistic durations, and potential delays before they impact project delivery.
الوظيفة:
يقوم بمراجعة الجداول الزمنية للمشاريع وتحديد أنشطة المسار الحرج والأنشطة القريبة من المسار الحرج ومخاطر الفلوات والعلاقات المفقودة بين الأنشطة ونقاط الضعف في الجدول الزمني والمدد غير الواقعية والتأخيرات المحتملة قبل تأثيرها على تنفيذ المشروع.
Typical Output Format:
•	Schedule Review Summary
•	Critical Path Activities
•	Near-Critical Activities
•	Float Risk Analysis
•	Missing Logic / Open Ends
•	Schedule Weaknesses
•	Recommended Corrective Actions
تنسيق المخرجات المتوقع:
•	ملخص مراجعة الجدول الزمني
•	أنشطة المسار الحرج
•	الأنشطة القريبة من المسار الحرج
•	تحليل مخاطر الفلوات
•	العلاقات المفقودة أو الأنشطة المفتوحة
•	نقاط ضعف الجدول الزمني
•	الإجراءات التصحيحية المقترحة

Typical Output Format:
•	Delay Event Summary
•	Cause of Delay
•	Impacted Activities
•	Critical Path Impact

تنسيق المخرجات المتوقع:

12. /CWICR Crew Optimizer
محسن الفرق التشغيلية
Function:
Uses labor productivity norms and project requirements to forecast workforce demand, optimize crew sizes, balance labor loading, create manpower forecasts, and improve resource planning throughout the project lifecycle.
الوظيفة:
يعتمد على معايير الإنتاجية ومتطلبات المشروع للتنبؤ باحتياجات العمالة وتحسين أحجام الفرق التشغيلية وموازنة الأحمال البشرية وإعداد توقعات الموارد البشرية وتحسين التخطيط للموارد خلال دورة حياة المشروع.
Typical Output Format:
•	Crew Optimization Summary
•	Required Manpower Forecast
•	Labor Loading Curve
•	Crew Size Recommendations
•	Trade Breakdown
•	Workforce Planning Notes
تنسيق المخرجات المتوقع:
•	ملخص تحسين الفرق التشغيلية
•	توقعات العمالة المطلوبة
•	منحنى تحميل العمالة
•	توصيات أحجام الفرق
•	تفصيل العمالة حسب التخصص
•	ملاحظات تخطيط الموارد البشرية
 
Purpose
الهدف
The goal of these AI tools is to reduce time spent on repetitive documentation, improve risk identification during tendering and project execution, strengthen schedule and resource management, standardize project documentation, and increase efficiency across estimating, contracts, project controls, project management, planning, and site engineering functions.
الهدف من هذه الأدوات هو تقليل الوقت المستغرق في الأعمال الورقية المتكررة، وتحسين اكتشاف المخاطر أثناء مرحلة المناقصات وتنفيذ المشاريع، وتعزيز إدارة الجداول الزمنية والموارد، وتوحيد المستندات والتقارير، وزيادة الكفاءة في أعمال التسعير والعقود والضبط والتحكم بالمشاريع وإدارة المشاريع والتخطيط والهندسة الموقعية.
 
Important Note
ملاحظة مهمة
These tools are intended to support engineers, project managers, quantity surveyors, planners, and contracts teams. They do not replace professional judgment. All AI-generated outputs must be reviewed and approved by the responsible project team before official use or submission.
تهدف هذه الأدوات إلى دعم المهندسين ومديري المشاريع ومهندسي الكميات والمخططين وفرق العقود، ولا تحل محل الحكم والخبرة المهنية. يجب مراجعة واعتماد جميع المخرجات الناتجة عن الذكاء الاصطناعي من قبل فريق المشروع المسؤول قبل استخدامها أو تقديمها بشكل رسمي.
